# Order Routing Engine

## Problem

Commerce platforms operate multiple fulfilment nodes including warehouses, stores and vendor locations.

Selecting the optimal fulfilment node for each order is complex due to:

• inventory availability  
• delivery speed requirements  
• logistics costs  
• warehouse capacity  

## Solution

The Order Routing Engine determines the optimal fulfilment node using a scoring mechanism based on:

Inventory availability  
Delivery SLA  
Shipping cost  
Fulfilment node capacity  

## Impact

Improves delivery speed while reducing logistics costs.
